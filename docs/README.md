# Landing Page de MindFlashX

Este directorio contiene la landing page para la descarga de MindFlashX.

## Estructura

```
docs/
├── index.html       # Landing page principal
├── assets/          # Recursos (logo, imágenes)
│   └── logo.svg     # Logo de la aplicación (SVG)
└── README.md        # Este archivo
```

## Uso

1. **Agregar el logo**: Coloca tu logo transparente en `assets/logo.png`
2. **Personalizar contenido**: Edita `index.html` para ajustar textos y botones
3. **Desplegar**: Sube la carpeta `docs/` a tu hosting

## Colores Utilizados

- **Azul cian**: `#0ea5e9` y `#06b6d4` (gradiente)
- **Gris oscuro fondo**: `#1a1a1a`
- **Gris medio**: `#2a2a2a`
- **Gris claro texto**: `#a0a0a0`
- **Blanco**: `#f5f5f5`
