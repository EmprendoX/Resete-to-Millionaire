# Directorio de Audios

Este directorio contiene los archivos MP3 de las afirmaciones de audio predefinidas.

## Archivos esperados:

- `abundance.mp3` - Afirmación de Abundancia (~2 minutos)
- `focus.mp3` - Afirmación de Enfoque (~2 minutos)
- `health.mp3` - Afirmación de Salud (~2 minutos)

Los archivos de audio se empaquetan automáticamente con la aplicación al hacer el build.
