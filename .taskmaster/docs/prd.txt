Product Requirements Document (PRD v3.0)
SUBLIMINALX™ — Reprograma tu mente mientras trabajas, estudias o navegas
1. Visión General
Propósito:
Desarrollar una app multiplataforma que muestre afirmaciones subliminales o visibles sobre cualquier aplicación, ayudando al usuario a reprogramar su mente, reforzar creencias positivas y mejorar su estado mental mientras realiza otras tareas.
Slogan:
“Reprograma tu mente mientras trabajas. Elige tu enfoque. Crea tu realidad.”
2. Plataformas y Alcance
Plataforma	Nivel de capacidad	Descripción
Web/PWA	Limitado	Pruebas y promoción (no puede sobreponerse a otras apps).
Desktop (Electron)	Completo	Overlay real sobre cualquier aplicación. Versión Premium.
Android (Nativa)	Completo	Overlay global mediante permiso SYSTEM_ALERT_WINDOW.
iOS	Limitado	Sin overlay global; versión visible o audio subliminal.
3. Objetivos del Producto
Ofrecer un sistema subliminal ético y configurable.
Permitir a los usuarios elegir entre múltiples listas temáticas (abundancia, salud, autoestima, disciplina, etc.) o crear sus propias listas.
Permitir la visualización subliminal real (10–200 ms) o visible.
Sincronizar listas y configuraciones mediante Supabase.
Integrar suscripciones y pagos por Stripe.
Desplegar en escritorio, web y móvil.

5. Arquitectura Técnica
Componente	Tecnología	Descripción
Frontend	React + Tailwind + Vite	Interfaz UI principal.
Backend	Supabase	Auth, DB y storage.
Desktop	Electron.js	Overlay global y app instalable (.exe, .dmg).
Móvil Android	React Native / Kotlin	Overlay global (SYSTEM_ALERT_WINDOW).
Pagos	Stripe	Checkout, Webhooks, planes.
Audio IA	ElevenLabs API	Afirmaciones habladas.
Hosting Web	Netlify / Vercel	Landing + versión demo.
6. Estructura de Base de Datos (Supabase)
users
Campo	Tipo
id	uuid
email	text
plan	text (free/premium/lifetime)
created_at	timestamp
affirmation_lists
Campo	Tipo	Descripción
id	uuid	Identificador único
user_id	uuid	Propietario (null = global)
name	text	Nombre de la lista
category	text	Categoría
is_global	boolean	True = lista del sistema
created_at	timestamp	
affirmations
Campo	Tipo
id	uuid
list_id	uuid
text	text
created_at	timestamp
settings
Campo	Tipo	Descripción
user_id	uuid	Usuario
speed	int	Intervalo entre frases (ms)
flash_time	int	Tiempo visible (ms)
opacity	float	Transparencia (0–1)
color	text	Color
position_mode	text	Random o Fixed
active_lists	array(uuid)	Listas activas
is_subliminal	boolean	Modo subliminal activo
consents
Campo	Tipo	Descripción
user_id	uuid	Usuario
consent_text_version	text	Versión de texto aceptado
accepted_at	timestamp	Fecha
device_info	text	Info del dispositivo
